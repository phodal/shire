Shire 语法由三部分组成：

- Hobbit Hole：用于定义数据结构。
  - Pattern-Action：用于定义数据处理逻辑。
  - Condition Visible：用于定义可见性条件。
  - Output：用于定义输出。
  - Variable：用于定义过程变量。
- Template：用于定义 LLM 模板。
  - Shire Template：用于定义代码生成模板。
